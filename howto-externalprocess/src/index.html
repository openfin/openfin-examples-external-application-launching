This OpenFin app will run an .exe file packaged with the app.

Find a pdf file and paste the full path (from "C:\" to ".pdf") in the box below.

The first button will run the External Process without passing arguments (and will Error).

The second button will run the External Process with the arguments displayed as input and output arguments.

This will be the first argument passed to the External Process

The input pdf path will be displayed here.

and this will be the second argument passed to the External Process

The output url will be displayed here


The default arguments will cause the .exe to run with an Error, unless you hard code paramaters in the 'app.json' which will work on your machine.


If you have pasted a valid path to a .pdf file the .exe should run with the above aruments and generate HTML files in the same directory as the .pdf.
